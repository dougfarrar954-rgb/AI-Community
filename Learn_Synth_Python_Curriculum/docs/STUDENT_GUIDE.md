# Student Guide

Welcome to the **Synth-Tutor** Python course!

## Your Path to Mastery
This course is divided into 13 modules, moving from the physics of a single sine wave to the complexity of a generative synthesizer.

## How to Learn
1. **Read the Module Master**: Each module has a markdown file in the `curriculum/` folder.
2. **Run the Examples**: Corresponding code is in the `examples/` folder.
3. **Talk to the Agent**: Use the instructions in the curriculum to prompt the agent for feedback.

## Tips for Success
- **Ear Protection**: Keep your volume low when first running new synthesis scripts.
- **Experiment**: Change variables and listen to the result immediately.
