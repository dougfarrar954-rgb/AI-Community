# Antigravity Setup Guide

## Requirements
- Python 3.8+
- Visual Studio Code (recommended)
- Antigravity extension installed

## Environment Initialization
1. Clone this repository.
2. Run `pip install numpy sounddevice scipy`.
3. Open the folder in your Antigravity instance.

## Connecting the Agent
Point your Antigravity agent to the `prompts/synth_tutor_system.md` file to initialize the **Synth-Tutor** persona.
